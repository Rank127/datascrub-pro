## Appendix A: Corporate Sales Playbook

### Ideal Customer Profile (ICP)

| Attribute | Target |
|-----------|--------|
| Company Size | 50-500 employees |
| Industries | Healthcare, Finance, Legal, Tech, Government Contractors |
| Decision Maker | CISO, IT Director, HR Director, COO |
| Pain Points | Phishing attacks, social engineering, compliance, employee safety |
| Budget | $5K-50K/year for security tools |

### Sales Process

1. **Lead Generation**
   - LinkedIn outreach
   - Content marketing (whitepapers, webinars)
   - Partner referrals
   - Trade shows (RSA, Black Hat)

2. **Discovery Call (15 min)**
   - Understand security stack
   - Identify pain points
   - Qualify budget and timeline

3. **Demo (30 min)**
   - Show admin dashboard
   - Run live scan on prospect's name
   - Demonstrate removal workflow
   - ROI calculator

4. **Proposal & Negotiation**
   - Custom pricing for 100+ seats
   - Annual contracts preferred
   - Net 30 payment terms

5. **Onboarding**
   - Dedicated success manager
   - Employee enrollment
   - Integration setup (SSO, API)

### Objection Handling

| Objection | Response |
|-----------|----------|
| "We already have identity protection" | "Traditional identity protection monitors AFTER theft. We PREVENT it by removing the data attackers use for social engineering." |
| "Too expensive" | "One successful phishing attack costs $50K+ on average. We're less than $10/employee/month." |
| "Employees can do this themselves" | "Each removal takes 20-30 minutes. With 2,000+ brokers, that's 1,000+ hours per employee. We automate it." |
| "Is this really a security issue?" | "90% of successful attacks start with reconnaissance. Your employees' personal data IS your attack surface." |
